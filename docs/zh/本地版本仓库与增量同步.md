## 一、仓库结构

```
.minecraft/
└── starter_repo/
    ├── repo.json                ← 根元信息
    ├── current → snapshots/v1.0.0  ← 当前快照 symlink
    ├── snapshots/
    │   └── v1.0.0/
    │       ├── manifest.json    ← 文件清单（hash/size）
    │       └── meta.json        ← 版本元信息（时间/来源/MC版本）
    └── files/
        └── {hash_prefix}/
            └── {sha256}         ← 按 hash 去重的文件
```

## 二、已完成功能

| 功能 | 文件 | 说明 |
|------|------|------|
| CreateFullSnapshot | repo.go | 扫描 mods/config，SHA256 清单 + 文件缓存 |

| DiffSinceSnapshot | incr_sync.go | 自定义路径扫描（修复 ScanDirectory bug）|
| CleanOrphaned | repo.go | 清理快照未引用的文件 |
