# MC 版本更新器 — 修复工具与备份恢复

> 补充方案：当 MC 无法正常启动时，一键修复环境

---

## 一、设计思路

### 核心原则

```
修复 = 把客户端状态"擦"回服务端定义的纯净状态
备份 = 擦之前保存用户所有自定义内容，支持随时恢复
```

### 触发场景

| 场景 | 用户操作 | 修复做了什么 |
|---|---|---|
| 更新后 MC 闪退 | 双击 starter.exe → `[F] 修复` | 删除所有模组 → 重新下载 → 覆盖配置 |
| 加了不兼容模组启动不了 | `starter repair` | 恢复纯净状态 |
| 配置文件改坏了 | `starter repair` | 覆盖所有配置文件 |
| 版本升级后崩了 | 自动检测到闪退 → 提示修复 | 全量同步 + 备份旧文件 |
| 单纯想重置整合包 | `starter repair --clean` | 清空一切，重新同步 |

## 二、修复命令

### 2.1 CLI 设计

```bash
# 一键修复（交互模式）
starter repair
  → 显示当前状态 → 询问要修复什么 → 执行

# 完全修复（不询问，全量重置）
starter repair --clean
  → 清空 mods/ → 重新下载
  → 覆盖 config/（force）
  → 重新安装 Fabric

# 仅修复模组
starter repair --mods-only

# 仅修复配置
starter repair --config-only

# 仅重新安装 Loader
starter repair --loader-only

# 回滚到上一个备份
starter repair --rollback
```

### 2.2 修复流程

```go
func Repair(cfg *RepairConfig, mcDir string) error {
    // Phase 0: 创建备份（修复前）
    backupPath, err := CreateBackup(mcDir)
    if err != nil {
        return fmt.Errorf("创建备份失败: %w", err)
    }
    fmt.Printf("✅ 已备份当前状态到: %s\n", backupPath)
    
    // Phase 1: 清理
    if cfg.CleanMods || cfg.CleanAll {
        backupDir(filepath.Join(mcDir, "mods"))
        os.RemoveAll(filepath.Join(mcDir, "mods"))
        os.MkdirAll(filepath.Join(mcDir, "mods"), 0755)
        fmt.Println("  ✓ 已清理 mods/")
    }
    
    if cfg.CleanConfig || cfg.CleanAll {
        backupDir(filepath.Join(mcDir, "config"))
        os.RemoveAll(filepath.Join(mcDir, "config"))
        os.MkdirAll(filepath.Join(mcDir, "config"), 0755)
        fmt.Println("  ✓ 已清理 config/")
    }
    
    if cfg.CleanAll {
        // 清理 resourcepacks/ 和 shaderpacks/
        backupDir(filepath.Join(mcDir, "resourcepacks"))
        backupDir(filepath.Join(mcDir, "shaderpacks"))
        fmt.Println("  ✓ 已清理 resourcepacks/ 和 shaderpacks/")
    }
    
    // Phase 2: 重新同步
    if cfg.ReinstallLoader || cfg.CleanAll {
        installFabric(mcDir, cfg.MCVersion, cfg.LoaderVersion)
        fmt.Println("  ✓ 已重新安装 Fabric")
    }
    
    // 下载模组（全量）
    downloadAllMods(cfg.ModList, filepath.Join(mcDir, "mods"))
    fmt.Println("  ✓ 已重新下载所有模组")
    
    // 覆盖配置（force 模式）
    syncConfigs(cfg.ConfigList, mcDir, "force")
    fmt.Println("  ✓ 已覆盖所有配置文件")
    
    // Phase 3: 重置同步标记
    resetSyncState()
    
    fmt.Println("========================")
    fmt.Println("✅ 修复完成！")
    fmt.Println("📦 备份位置:", backupPath)
    fmt.Println("💡 如需恢复请运行: starter repair --rollback")
    
    return nil
}
```

## 三、备份系统

### 3.1 备份内容

| 目录 | 是否备份 | 说明 |
|---|---|---|
| mods/ | ✅ 必备份 | 用户加的模组、改的文件 |
| config/ | ✅ 必备份 | 用户改过的配置 |
| resourcepacks/ | ✅ 必备份 | 用户加的资源包 |
| shaderpacks/ | ✅ 必备份 | 用户加的光影包 |
| saves/ | ✅ 必备份 | 存档（非常重要！） |
| screenshots/ | ✅ 必备份 | 截图 |
| options.txt | ✅ 必备份 | MC 设置 |
| versions/ | ❌ 不备份 | 太大，可从服务端重新下载 |
| assets/ | ❌ 不备份 | 太大 |
| libraries/ | ❌ 不备份 | 太大 |

### 3.2 备份存储

```go
type BackupMeta struct {
    ID          string    `json:"id"`           // 时间戳，如 "20260429_022800"
    CreatedAt   time.Time `json:"created_at"`
    Reason      string    `json:"reason"`       // "repair" / "upgrade" / "manual"
    MCVersion   string    `json:"mc_version"`
    ModCount    int       `json:"mod_count"`
    ConfigCount int       `json:"config_count"`
    SaveCount   int       `json:"save_count"`
    SizeBytes   int64     `json:"size_bytes"`
}
```

**存储结构**：
```
.minecraft/
├── mods/                         ← 当前使用的模组
├── config/                       ← 当前使用的配置
├── ...
└── starter_backups/              ← 备份目录
    ├── backup_20260429_022800/   ← 修复前自动创建
    │   ├── meta.json             ← 备份元信息
    │   ├── mods/                 ← 模组备份
    │   ├── config/               ← 配置备份
    │   ├── saves/                ← 存档备份
    │   ├── resourcepacks/
    │   ├── shaderpacks/
    │   ├── screenshots/
    │   └── options.txt
    ├── backup_20260428_153000/
    └── backup_20260427_120000/
```

### 3.3 自动备份 vs 手动备份

**自动备份触发条件**：
1. `repair` 命令执行前 → 自动创建备份
2. MC 大版本升级（如 1.20.1 → 1.21）→ 自动创建备份
3. 首次 sync 前（如果已有 `.minecraft`）→ 自动创建备份

**保留策略**：
```
最多保留 5 个备份
超过 5 个时删除最旧的
备份总大小超过 2GB 时提示清理
```

### 3.4 备份实现

```go
func CreateBackup(mcDir string) (string, error) {
    backupDir := filepath.Join(mcDir, "starter_backups")
    os.MkdirAll(backupDir, 0755)
    
    // 生成备份 ID
    id := time.Now().Format("20060102_150405")
    backupPath := filepath.Join(backupDir, "backup_"+id)
    os.MkdirAll(backupPath, 0755)
    
    // 计算需要备份的目录
    items := []struct{
        src string
        dst string
        required bool
    }{
        {filepath.Join(mcDir, "mods"),            filepath.Join(backupPath, "mods"),            false},
        {filepath.Join(mcDir, "config"),          filepath.Join(backupPath, "config"),          false},
        {filepath.Join(mcDir, "saves"),           filepath.Join(backupPath, "saves"),           false},
        {filepath.Join(mcDir, "resourcepacks"),   filepath.Join(backupPath, "resourcepacks"),   false},
        {filepath.Join(mcDir, "shaderpacks"),     filepath.Join(backupPath, "shaderpacks"),     false},
        {filepath.Join(mcDir, "screenshots"),     filepath.Join(backupPath, "screenshots"),     false},
        {filepath.Join(mcDir, "options.txt"),     filepath.Join(backupPath, "options.txt"),     true},
    }
    
    modCount := 0
    configCount := 0
    saveCount := 0
    var totalSize int64
    
    for _, item := range items {
        if _, err := os.Stat(item.src); os.IsNotExist(err) {
            if item.required {
                return "", fmt.Errorf("需要备份的文件不存在: %s", item.src)
            }
            continue
        }
        
        // 复制目录/文件
        err := copyRecursive(item.src, item.dst, func(path string, info os.FileInfo) {
            totalSize += info.Size()
            if strings.Contains(path, "mods") && strings.HasSuffix(path, ".jar") {
                modCount++
            }
            if strings.Contains(path, "config") {
                configCount++
            }
            if strings.Contains(path, "saves") {
                saveCount++
            }
        })
        if err != nil {
            return "", fmt.Errorf("备份失败 %s: %w", item.src, err)
        }
    }
    
    // 写入元数据
    meta := BackupMeta{
        ID:          id,
        CreatedAt:   time.Now(),
        Reason:      "repair",
        ModCount:    modCount,
        ConfigCount: configCount,
        SaveCount:   saveCount,
        SizeBytes:   totalSize,
    }
    metaBytes, _ := json.MarshalIndent(meta, "", "  ")
    os.WriteFile(filepath.Join(backupPath, "meta.json"), metaBytes, 0644)
    
    // 清理旧备份
    cleanupOldBackups(backupDir, 5)
    
    return backupPath, nil
}
```

### 3.5 删除旧备份

```go
func cleanupOldBackups(backupDir string, maxKeep int) {
    entries, _ := os.ReadDir(backupDir)
    
    var backups []os.DirEntry
    for _, e := range entries {
        if e.IsDir() && strings.HasPrefix(e.Name(), "backup_") {
            backups = append(backups, e)
        }
    }
    
    // 按名称排序（名称是时间戳，天然有序）
    sort.Slice(backups, func(i, j int) bool {
        return backups[i].Name() < backups[j].Name()
    })
    
    // 删掉最旧的
    for len(backups) > maxKeep {
        old := backups[0]
        os.RemoveAll(filepath.Join(backupDir, old.Name()))
        log("已删除旧备份: %s", old.Name())
        backups = backups[1:]
    }
}
```

## 四、恢复

### 4.1 恢复回滚

```go
func Rollback(mcDir string) error {
    backupDir := filepath.Join(mcDir, "starter_backups")
    
    // 列出可用备份
    backups := listBackups(backupDir)
    if len(backups) == 0 {
        return fmt.Errorf("没有可用的备份")
    }
    
    // 显示备份列表
    fmt.Println("可用的备份：")
    for i, b := range backups {
        fmt.Printf("  %d. %s (%s) - %s\n", i+1, b.ID, 
            b.CreatedAt.Format("2006-01-02 15:04:05"), b.Reason)
    }
    
    // 选择要恢复的备份
    idx := promptSelect("选择要恢复的备份 (1-%d):", len(backups))
    selected := backups[idx-1]
    
    // 恢复前的二次确认
    if !promptConfirm("恢复将覆盖当前 mods/、config/、saves/ 等目录，是否继续？") {
        return nil
    }
    
    // 执行恢复
    restorePath := filepath.Join(backupDir, "backup_"+selected.ID)
    
    // 先备份当前状态（防手滑）
    tempBackup, _ := CreateBackup(mcDir)
    fmt.Printf("已备份当前状态到: %s\n", tempBackup)
    
    // 恢复各目录
    dirs := []string{"mods", "config", "saves", "resourcepacks", "shaderpacks", "screenshots"}
    for _, dir := range dirs {
        src := filepath.Join(restorePath, dir)
        dst := filepath.Join(mcDir, dir)
        
        if _, err := os.Stat(src); os.IsNotExist(err) {
            continue
        }
        
        os.RemoveAll(dst)
        copyRecursive(src, dst, nil)
    }
    
    // 恢复 options.txt
    optionsSrc := filepath.Join(restorePath, "options.txt")
    optionsDst := filepath.Join(mcDir, "options.txt")
    if _, err := os.Stat(optionsSrc); err == nil {
        copyFile(optionsSrc, optionsDst)
    }
    
    fmt.Println("✅ 恢复完成！")
    fmt.Println("💡 如需撤销恢复，可使用最新备份再次恢复")
    
    return nil
}
```

### 5.1 启动失败检测

```go
// 在 launch 命令中集成自动检测
func LaunchWithAutoRepair(cfg *Config, mcDir string) error {
    // 记录启动尝试
    attempt := loadLaunchAttempts(mcDir)
    
    // 检测上次启动是否正常退出
    if attempt.LastExitCode != 0 && attempt.LastExitCode != -1 {
        // 上次退出码不为 0，可能崩了
        fmt.Println("⚠  检测到上次 Minecraft 异常退出")
        fmt.Println("   退出码:", attempt.LastExitCode)
        
        if promptConfirm("是否尝试修复？修复将清空模组和配置并重新下载") {
            Repair(&RepairConfig{CleanAll: true}, mcDir)
        }
    }
    
    // 启动 MC 并监控
    cmd := launchMinecraft(cfg, mcDir)
    err := cmd.Run()
    
    // 记录本次退出码
    attempt.LastExitCode = cmd.ProcessState.ExitCode()
    attempt.LastLaunchTime = time.Now()
    saveLaunchAttempts(mcDir, attempt)
    
    return err
}
```

### 5.2 崩溃日志检测

```go
func checkCrashReports(mcDir string) bool {
    crashDir := filepath.Join(mcDir, "crash-reports")
    if _, err := os.Stat(crashDir); os.IsNotExist(err) {
        return false
    }
    
    entries, _ := os.ReadDir(crashDir)
    for _, e := range entries {
        if strings.HasSuffix(e.Name(), ".txt") && 
           isRecentlyModified(filepath.Join(crashDir, e.Name()), 24*time.Hour) {
            return true  // 24 小时内有崩溃报告
        }
    }
    return false
}
```

## 七、与正常 sync 的区别

| | `starter sync`（正常同步） | `starter repair`（修复） |
|---|---|---|
| 模组 | 差异同步 | 全量重新下载 |
| 配置 | once / force | force（全部覆盖） |
| 用户自定义模组 | 保留 | 删除 + 备份 |
| 备份 | 不备份 | 自动创建完整备份 |
| 触发条件 | 用户主动运行 | 用户修复 / 自动检测故障 |
| 风险 | 低 | 高（但可回滚） |
