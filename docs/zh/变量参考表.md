## 1. 模型层 — `internal/model/types.go`

## 3. 启动器内核 — `internal/launcher/`

### 3.1 Updater

**文件**: `internal/launcher/update.go`

**职责**：客户端增量更新管理器。整合包同步引擎，适配 C/S + 多包模式。
从服务端拉增量清单 → 按 hash 下载文件 → 写 CacheStore。

**字段**：

| 字段 | 类型 | 说明 |
|------|------|------|
| `cfgDir` | `string` | 配置目录 |
| `mcDir` | `string` | .minecraft 目录 |
| `cache` | `*CacheStore` | 文件缓存 |
| `repo` | `*LocalRepo` | 本地仓库 |
| `api` | `*config.Manager` | API 客户端 |

**方法**：

| 方法 | 签名 | 说明 |
|------|------|------|
| `NewUpdater` | `func NewUpdater(cfgDir, mcDir string, api *Manager) *Updater` | 创建更新管理器 |
| `UpdatePack` | `func (u *Updater) UpdatePack(serverURL, packName string, packState *PackState, forceFull bool) (*UpdateResult, error)` | 更新单个包 |
| `UpdateAllPacks` | `func (u *Updater) UpdateAllPacks(serverURL string, packs map[string]PackState, onChannels func(...)) map[string]*UpdateResult` | 批量更新 |
| `EnsureVersion` | `func (u *Updater) EnsureVersion(req EnsureRequest) (*EnsureResult, error)` | 确保 MC+Loader 已安装 |
| `CacheStore` | `func (u *Updater) CacheStore() *CacheStore` | 返回内部缓存 |
| `LocalRepo` | `func (u *Updater) LocalRepo() *LocalRepo` | 返回内部仓库 |
| `DownloadFullPack` | `func (u *Updater) DownloadFullPack(fullPackURL, fullPackHash, destDir string) (*UpdateResult, error)` | 全量下载 |
