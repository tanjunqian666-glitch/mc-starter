## 二、核心问题详解

### 🔴 A. versionName 变量遮蔽（Critical Bug）

**文件**: `cmd/starter/main.go` `run()` 函数（第 368-374 行）

**当前代码**（出错逻辑）：
```go
// 第 368 行：首次声明
versionName := fmt.Sprintf("mc-starter-%s", packName)
// 第 369 行：用首次声明的 versionName 构建目录
versionTargetDir := filepath.Join(versionsDir, versionName)

// 第 372 行：重新声明（shadowing！）
// !!! 这创建了第二个局部变量 versionName
versionName := fmt.Sprintf("mc-starter-%s", packName)
if detail != nil && detail.DisplayName != "" {
    // 第 374 行：只修改了第二个局部变量
    versionName = detail.DisplayName
}
```

**Bug 后果**：
- `versionTargetDir` 永远指向 `versions/mc-starter-{packName}/`
- 即使 `detail.DisplayName` 返回了自定义显示名（如"主服整合包"），
  也不会更新 `versionTargetDir`
- 结果：写入的 `version.json` 在 `versions/mc-starter-主服整合包/`，
  但实际写入的目录是 `versions/mc-starter-{packName}/`
- PCL2 读取时找不到匹配的版本配置

**修复方案**：
```go
// 删除第 372 行的重复声明，改为赋值
versionName = fmt.Sprintf("mc-starter-%s", packName)
if detail != nil && detail.DisplayName != "" {
    versionName = detail.DisplayName
}
```

### 🟡 D. `fmt.Scanln` 在非交互模式会挂起

## 三、变量遮蔽 Bug 详细影响评估

### 3.1 触发条件

触发条件：包有 `DisplayName` 且与默认的 `mc-starter-{packName}` 不同。

正常情况（`detail.DisplayName == ""`）：
- `versionName` = `mc-starter-{packName}`（两次声明后值相同）
- `versionTargetDir` = `versions/mc-starter-{packName}/`
- ✅ 无异常

异常情况（`detail.DisplayName = "主服整合包"`）：
- 第一次声明：`versionName` = `mc-starter-main`；`versionTargetDir` = `versions/mc-starter-main/`
- 第二次声明：`versionName`（新变量）= `mc-starter-main` → 被改为 `主服整合包`
- `version.json` 写入：`WriteVersionMetaJSON(versionTargetDir, versionName, ...)` 中
  - `versionTargetDir` = `versions/mc-starter-main/` ❌
  - `versionName` = `主服整合包`
- 结果：`versions/mc-starter-main/主服整合包.json` 包含了错误的 `id: "主服整合包"`，
  但目录名仍是 `mc-starter-main`。PCL2 在读取时取目录名和 id 不匹配，可能导致加载失败。

### 3.2 影响范围

- `starter run` 命令中处理所有启用的包
- 影响所有有非空 `DisplayName` 的包的版本 JSON 写入
- 影响合并后的版本在启动器中的识别

### 3.3 修复确认

✅ 第 372 行 `versionName :=` → `versionName =`
修复后逻辑：
```go
versionName := fmt.Sprintf("mc-starter-%s", packName)  // 首次声明
versionTargetDir := filepath.Join(versionsDir, versionName)  // 用初始值

// 重新赋值（非重新声明），如果 versionTargetDir 需要改变，需要重新计算
versionName = fmt.Sprintf("mc-starter-%s", packName)  // 或直接 = ""
if detail != nil && detail.DisplayName != "" {
    versionName = detail.DisplayName
}
// ⚠️ 注意：versionTargetDir 需要根据新的 versionName 重新计算！
```
