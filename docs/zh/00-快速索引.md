## GUI 关键方法速查

| 方法 | 位置 | 作用 |
|------|------|------|
| `NewViewModel(cfgDir)` → `vm.Init()` | `viewmodel.go` | 加载配置+版本列表，返回是否首次运行 |
| `vm.PackNames()` | `viewmodel.go` | 下拉框显示名列表（仅已启用版本） |
| `vm.ServerPacksFiltered()` | `viewmodel.go` | 返回已启用的 PackInfo 列表 |
| `vm.SelectPack(name)` | `viewmodel.go` | 切换选中版本+更新本地/最新版本号 |
| `vm.CurrentPackStatus()` | `viewmodel.go` | 计算按钮文案/可用性/状态栏颜色 |
| `vm.MarkSyncStart/Done/Error/Cancelled()` | `viewmodel.go` | 同步生命周期（自动穿透状态机） |
| `vm.CanSync()` / `vm.SyncType()` | `viewmodel.go` | 判断是否可同步 + install/update |
| `NewOrchestrator(cfgDir, vm, eb)` | `orchestrator.go` | 流程调度器 |
| `orc.UpdateOrInstall()` | `orchestrator.go` | 安装/更新当前选中版本（goroutine） |
| `orc.DoRepairCleanAll/MC/ModsSync()` | `orchestrator.go` | 各修复操作（goroutine） |
| `orc.DoCrashLogUpload(mcDir)` | `orchestrator.go` | 崩溃日志上传（goroutine） |
| `StateMachine.Transition(to)` | `state.go` | 尝试状态转换，无效转换返回 false |
| `StateMachine.IsBusy()` | `state.go` | 是否在忙碌（非 Idle/Error/Done/Cancelled） |

## GUI 数据流

```
用户点按钮 → app.go → Orchestrator.Method()
  → Orchestrator 开 goroutine 执行
  → 发射 EventBus 事件 (Progress/StateChange/Error/Log)
  → eventLoop() 收到事件 → mw.Synchronize() → refreshUI()
```

关键状态：`PackNames()` 只返回 `Enabled==true` 的版本。设置页勾选副版本后，关联 MC 目录行通过 SetVisible 显隐。
