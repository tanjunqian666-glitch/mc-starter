    2. 拉服务端包列表，获取 (mc_version, loader)

    4. UpdatePack → 下自定义内容到 packs/
    5. MergePackToVersion → 合并到 versions/<name>/
    6. 完成提示
- **`handleUpdate`/`handleUpdateMulti` 增强**
  - 单包/批量更新完成后，拉取包详情显示 MC 版本 + Loader 信息
